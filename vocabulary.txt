
1) unit step function - функция единичного скачка, или единичная ступенчатая функция.

        f(z) = 1   if z >= 0
        f(z) = -1  if z < 0

2) learning rate - n темп обучения (константа между 0.0 и 1.0)


3) yi - true class label, истинная метка класса для i-го объекта

4) y^i - predicted class label, идентифицированная метка класса для i-го объекта (с помощью unit step function)

5) perceptron learning rule - правило обучения перцептрона

        delta_wj = n * ( yi - y^i ) * xij

        используется для обновления веса wj
        получаем разницу между истинной меткой класса и идентифицированной меткой класса меткой класса
        умножаем на темп обучения
        полученную разницу умножаем на значение j-го параметра i-го объекта
